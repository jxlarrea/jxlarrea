## Hi there 👋

I build software and I like contributing to free software and open source in general in my spare time.

I'm obsessed with Home Assistant Voice and the idea that one day it'll make Alexa and Google Home obsolete. Here's what I'm building toward that:

- [Voice Satellite](https://github.com/jxlarrea/voice-satellite-card-integration): Turns any tablet, phone, or browser into a hands-free voice assistant for Home Assistant
- [LLM Tools](https://github.com/jxlarrea/voice-satellite-card-llm-tools): Extends Voice Satellite with weather, financial, image and video search capabilities.
- [Wyoming Voice Match](https://github.com/jxlarrea/wyoming-voice-match): A Wyoming protocol ASR proxy that verifies speaker identity and isolates voice commands from background noise before forwarding audio to a downstream speech-to-text service.
- [OpenWakeWord Trainer](https://github.com/jxlarrea/openwakeword-trainer): A Dockerized, GPU-accelerated trainer for openWakeWord custom models with a Web UI. It is built especially for people using Voice Satellite, where wake words usually need to work from wall-mounted tablets instead of dedicated microphone arrays. 
- [Home Assistant Voice Control Recipes](https://github.com/jxlarrea/homeassistant-voice-recipes): GPU/CUDA-accelerated voice control stack for Home Assistant. Runs on x86/x64 and ARM64 (including the NVIDIA DGX Spark). 100% Local - No Cloud, No Subscriptions.
